# Requirements

1. vboxnet1 should be exist with enable DHCP server.
2. Version VirtualBox kernel modules do match this version of VirtualBox:

- Run modinfo vboxdrv.
- Run apt-cache policy virtualbox or yum list virtualbox
- Version should be equals
